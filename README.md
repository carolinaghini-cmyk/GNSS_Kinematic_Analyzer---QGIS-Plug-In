# GNSS_Kinematic_Analyzer---QGIS-Plug-In
The plug-in processes kinematic GNSS trajectories acquired with dual-antenna, dual-VRS configurations. It converts raw PPKTrack or RTKLib files into  QGIS layers, computes VRS consensus positions, evaluates the inter-receiver baseline geometry, and identifies static segments for precision assessment. Developed by DICEA — Sapienza University.
