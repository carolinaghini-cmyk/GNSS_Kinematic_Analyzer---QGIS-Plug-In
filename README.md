# GNSS_Kinematic_Analyzer - QGIS-Plug-In
GNSS Kinematic Analyzer processes kinematic GNSS trajectories acquired with dual-antenna, dual-VRS configurations. It converts raw PPKTrack (.txt) or RTKLib (.pos) files into georeferenced QGIS layers, computes per-epoch VRS consensus positions, evaluates the inter-receiver baseline geometry, and identifies static segments for precision assessment. All timestamps are handled in UTC regardless of the input time system.

Developed by the Department of Civil, Building and Environmental Engineering (DICEA) — Sapienza University of Rome.

Authors:	Silvia Di Giannantonio, Carolina Ghini, Augusto Mazzoni
